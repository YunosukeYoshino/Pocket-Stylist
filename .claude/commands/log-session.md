# Session Logging Command

セッション終了時の自動ログ記録を実行します。

## 実行内容

1. **現在の日付でログファイル作成**
   - `logs/YYYY-MM-DD-session-log.md`形式
   - 既存ファイルがある場合は追記

2. **プロジェクト状況の記録**
   - セッション開始・終了時刻
   - 変更されたファイル一覧
   - docs/設計書の状況
   - 主要な作業内容

3. **Git状況の記録**（リポジトリの場合）
   - 変更ファイル状況
   - コミット履歴
